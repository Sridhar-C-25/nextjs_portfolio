## Getting Started

First, run the development server:

You don't have pnpm i want install globaly:

```bash
npm i -g pnpm
```

```bash
pnpm dev
```

Overall install:

```bash
pnpm i
```

install packages:

```bash
pnpm add <your-package-name>
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
